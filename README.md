# Odin Recipes

This project is part of The Odin Project curriculum. It is a basic website that showcases a collection of recipes using only HTML.

## Project Description

The goal of this project is to build a simple multi-page website that links to different recipe pages. Each page will include ingredients, steps, and basic structure using HTML.

## Skills Demonstrated

By completing this project, I will have demonstrated:

- Basic HTML structure (boilerplate, headings, paragraphs)
- Working with links and navigation between pages
- Using lists (ordered and unordered)
- Adding images to a webpage
- Organizing files and folders properly
- Writing clean and readable code
